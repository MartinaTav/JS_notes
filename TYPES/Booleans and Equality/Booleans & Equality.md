# Booleans & Equality
True or false
we use them for the logic (if statements ...)
They can be manually set or calculated

```javascript
let isDrawing = false;

cont age = 18;
const ofAge = age > 19;
console.log(ofAge);
// false
```

```javascript
age === 18;
// true
age === 100;
false
```
```===``` will check value and type
```==``` will only check for value
